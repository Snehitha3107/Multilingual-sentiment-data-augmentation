# Data Augmentation for Multimodal Multilingual Sentiment Analysis

## Project Overview

This project investigates data augmentation techniques for **multimodal multilingual sentiment analysis** by combining traditional augmentation methods with **Large Language Model (LLM)-based synthetic data generation**.

The objective is to improve dataset quality, class balance, and linguistic diversity while preserving sentiment consistency for downstream sentiment classification tasks. The project evaluates both classical and LLM-driven augmentation strategies using statistical analysis and validation metrics.

This project was completed as part of the **Statistical Thinking for AI (STTAI)** course at **Indian Institute of Technology Gandhinagar**.

---

## Task 1: Dataset Analysis

The original multilingual movie review dataset was analyzed to understand sentiment distribution and identify class imbalance.

The analysis included:

- Class distribution visualization
- Dataset statistics
- Identification of underrepresented sentiment classes

---

## Task 2: Classical Data Augmentation

Implemented classical augmentation techniques to improve dataset diversity while preserving sentiment labels.

The augmented dataset was generated and compared with the original dataset to evaluate improvements in class balance.

---

## Task 3: LLM-based Data Generation

Large Language Models were used to generate multilingual movie reviews with controlled sentiment labels.

Prompt engineering techniques were applied to generate high-quality reviews while maintaining the required sentiment distribution.

Generated reviews were validated for:

- Sentiment consistency
- Label correctness
- Linguistic diversity
- Data quality

---

## Task 4: Dataset Evaluation

The augmented dataset was evaluated using statistical analysis and diversity metrics.

Performance analysis included:

- Class balance evaluation
- Diversity analysis
- Sentiment validation
- Quality assessment of generated reviews
